# Slider
<Description name="Slider" />

## Быстрый старт

```tsx live
import React, { useState } from 'react';
import { Slider } from '@salutejs/{{ package }}';

export function App() {
    const [value, setValue] = useState(30);

    return (
        <div style=\{{ width: '25rem', display: 'flex', flexDirection: 'column' }}>
            <Slider
                min={0}
                max={100}
                value={value}
                label="Громкость"
                onChange={(val) => setValue(val)}
                onChangeCommitted={(val) => setValue(val)}
            />
        </div>
    );
}
```

## Диапазон значений

Передайте массив из двух чисел в `value`, чтобы использовать двойной слайдер:

```tsx live
import React, { useState } from 'react';
import { Slider } from '@salutejs/{{ package }}';

export function App() {
    const [value, setValue] = useState([10, 80]);

    const sortValues = (values) =>
        values.map((val) => Math.min(100, Math.max(0, val))).sort((a, b) => a - b);

    return (
        <div style=\{{ width: '25rem', display: 'flex', flexDirection: 'column' }}>
            <Slider
                min={0}
                max={100}
                value={value}
                label="Цена товара"
                onChange={(values) => setValue(sortValues(values))}
                onChangeCommitted={(values) => setValue(sortValues(values))}
                onBlurTextField={(values) => setValue(sortValues(values))}
                onKeyDownTextField={(values, event) => {
                    if (event.key === 'Enter') setValue(sortValues(values));
                }}
            />
        </div>
    );
}
```

## Текущее значение

Свойство `showCurrentValue` включает отображение текущего значения над ползунком. Видимость регулируется через `currentValueVisibility`:

```tsx live
import React, { useState } from 'react';
import { Slider } from '@salutejs/{{ package }}';

export function App() {
    const [v1, setV1] = useState(50);
    const [v2, setV2] = useState(50);

    return (
        <div style=\{{ display: 'flex', gap: '2rem', flexDirection: 'column', width: '25rem' }}>
            <Slider
                min={0}
                max={100}
                value={v1}
                showCurrentValue
                currentValueVisibility="always"
                label="Всегда"
                onChange={setV1}
            />
            <Slider
                min={0}
                max={100}
                value={v2}
                showCurrentValue
                currentValueVisibility="hover"
                label="При наведении"
                onChange={setV2}
            />
        </div>
    );
}
```

## Шаг изменения

Свойство `step` задаёт шаг ползунка. Свойство `multipleStepSize` — увеличенный шаг при нажатии `PageUp` / `PageDown` (в процентах от диапазона):

```tsx live
import React, { useState } from 'react';
import { Slider } from '@salutejs/{{ package }}';

export function App() {
    const [v1, setV1] = useState(50);
    const [v2, setV2] = useState(5000);

    return (
        <div style=\{{ display: 'flex', gap: '2rem', flexDirection: 'column', width: '25rem' }}>
            <Slider min={0} max={100} step={5} value={v1} label="Шаг 5" showScale onChange={setV1} />
            <Slider min={0} max={10000} step={500} multipleStepSize={10} value={v2} label="Шаг 500, PageUp/Down — 10%" showScale onChange={setV2} />
        </div>
    );
}
```

## Состояния

Компонент поддерживает состояние `disabled`:

```tsx live
import React, { useState } from 'react';
import { Slider } from '@salutejs/{{ package }}';

export function App() {
    const [value, setValue] = useState(50);

    return (
        <div style=\{{ display: 'flex', gap: '1rem', flexDirection: 'column', width: '25rem' }}>
            <Slider min={0} max={100} value={value} label="Активный" onChange={setValue} />
            <Slider min={0} max={100} value={50} label="Неактивный" disabled />
        </div>
    );
}
```
