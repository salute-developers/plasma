# Steps
<Description name="Steps" />

## Быстрый старт

```tsx live
import React, { useState } from 'react';
import { Steps } from '@salutejs/plasma-homeds';

export function App() {
    const [items, setItems] = useState([{
        indicator: 1,
        title: 'Шаг 1',
        status: 'active',
    }, {
        indicator: 2,
        title: 'Шаг 2',
        status: 'inactive',
    }, {
        indicator: 3,
        title: 'Шаг 3',
        status: 'inactive',
    }]);

    const onChange = (item, index, prevIndex) => {
        if (prevIndex !== undefined) {
            items[prevIndex].status = 'completed';
        }

        items[index].status = 'active';

        setItems([...items]);
    };

    return (
        <section>
            <Steps items={items} onChange={onChange} />
        </section>
    );
}
```

## Внешний вид

<Tabs>
    <TabItem value="view" label="View" default>
        Свойство `view` применяется ко всему компоненту и задаёт цвет завершённых шагов и разделителя. Принимает значения `default` и `accent`.

        ```tsx live

        import { Steps } from '@salutejs/plasma-homeds';

        export function App() {
            const completedItems = [{
                indicator: 1,
                title: 'Title',
                status: 'completed',
            }, {
                indicator: 2,
                title: 'Title',
                status: 'active',
            }, {
                indicator: 3,
                title: 'Title',
                status: 'inactive',
            }];

            return (
                <div style={{ display: 'flex', gap: '2rem', flexDirection: 'column' }}>
                    <Steps view="default" items={completedItems} />
                    <Steps view="accent" items={completedItems} />
                </div>
            );
        }
        ```
    </TabItem>
    <TabItem value="itemView" label="ItemView">
        Свойство `itemView` применяется к индикатору и заголовку **активного** шага. Принимает значения `default`, `negative`, `warning`, `positive`.

        :::info
        Значения `itemView` применяются к шагу только когда его статус — `active`.
        :::

        ```tsx live
        import React from 'react';
        import { Steps } from '@salutejs/plasma-homeds';

        export function App() {
            const activeItems = [{
                indicator: 1,
                title: 'Active step',
                status: 'active',
            }, {
                indicator: 2,
                title: 'Inactive step',
                status: 'inactive',
            }];

            return (
                <div style={{ display: 'flex', gap: '2rem', flexDirection: 'column' }}>
                    <Steps itemView="default" items={activeItems} />
                    <Steps itemView="positive" items={activeItems} />
                    <Steps itemView="warning" items={activeItems} />
                    <Steps itemView="negative" items={activeItems} />
                </div>
            );
        }
        ```
    </TabItem>
    <TabItem value="completedItemView" label="CompletedItemView">
        Свойство `completedItemView` задаётся на каждом шаге через `items[n].completedItemView` и переопределяет внешний вид **завершённого** шага. Принимает значения `default`, `accent`, `negative`, `warning`, `positive`.

                title: 'positive',
                status: 'completed',
                completedItemView: 'positive',
            }, {
                indicator: 3,
                title: 'warning',
                status: 'completed',
                completedItemView: 'warning',
            }, {
                indicator: 4,
                title: 'negative',
                status: 'completed',
                completedItemView: 'negative',
            }, {
                indicator: 5,
                title: 'active',
                status: 'active',
            }];

            return (
                <section>
                    <Steps items={items} />

## Управление навигацией

Колбэк `onChange` вызывается при клике на шаг. Принимает `(item, index, prevIndex)`. Внутри него обновляйте статусы шагов вручную.

```tsx live
import React, { useState } from 'react';
import { Steps, StepItemProps } from '@salutejs/plasma-homeds';

export function App() {
    const [items, setItems] = useState([{
        indicator: 1,
        title: 'Title',
        content: 'Content',
        status: 'active',
    }, {
        indicator: 2,
        title: 'Title',
        content: 'Content',
        status: 'inactive',
    }, {
        indicator: 3,
        title: 'Title',
        content: 'Content',
        status: 'inactive',
    }]);

    const onChange = (item: StepItemProps, index: number, prevIndex: number) => {
        if (prevIndex !== undefined) {
            items[prevIndex].status = 'completed';
        }

        items[index].status = 'active';

        setItems([...items]);
    };

    return (
        <section>
            <Steps items={items} onChange={onChange} />
        </section>
    );
}
```

## Внешнее управление

Свойство `current` позволяет управлять активным шагом извне без использования `onChange`:

```tsx live
import React, { useState } from 'react';
import { Steps, Button } from '@salutejs/plasma-homeds';

export function App() {
    const items = [{
        indicator: 1,
    }, {
        indicator: 2,
    }, {
        indicator: 3,
    }];

    const [current, setCurrent] = useState(0);
    const onClick = () => setCurrent((prev) => prev + 1);

    return (
        <section>
            <Steps items={items} current={current} />
            <br />
            <Button view="secondary" onClick={onClick}>
                Next
            </Button>
        </section>
    );
}
```

## Кастомный размер шага

Каждому шагу можно задать индивидуальный размер через свойство `height` (вертикальный режим) или `width` (горизонтальный режим).

:::info
В вертикальном режиме последний шаг **не требует** явного `height` — он автоматически занимает минимально необходимое пространство (fit-content). Для остальных шагов `height` позволяет зарезервировать место под контент с переменной высотой (например, аккордеон).
:::

## Вложенные шаги с аккордеоном

Шаг может содержать произвольный контент, в том числе другой компонент `Steps`. Для корректного отображения передайте `height` внешним шагам, которые содержат контент с переменной высотой.

```tsx live
import React, { useState } from 'react';
import { Steps, Accordion, AccordionItem } from '@salutejs/plasma-homeds';

export function App() {
    const [open, setOpen] = useState(true);

    const [innerSteps, setInnerSteps] = useState([{
        indicator: 1,
        title: 'Установка зависимостей',
        status: 'active',
        height: '3.5rem',
    }, {
        indicator: 2,
        title: 'Настройка окружения',
        status: 'inactive',
        height: '3.5rem',
    }, {
        indicator: 3,
        title: 'Запуск проекта',
        status: 'inactive',
    }]);

    const [outerStatuses, setOuterStatuses] = useState([
        { status: 'completed' },
        { status: 'active' },
        { status: 'inactive' },
    ]);

    const handleInnerChange = (_item, idx, prevIdx) => {
        setInnerSteps(steps => {
            const next = [...steps];
            if (prevIdx !== undefined) {
                next[prevIdx] = { ...next[prevIdx], status: 'completed' };
            }
            next[idx] = { ...next[idx], status: 'active' };
            return next;
        });
    };

    const handleOuterChange = (_item, idx, prevIdx) => {
        setOuterStatuses(statuses => {
            const next = statuses.map(s => ({ ...s }));
            if (prevIdx !== undefined) {
                next[prevIdx] = { ...next[prevIdx], status: 'completed' };
            }
            next[idx] = { ...next[idx], status: 'active' };
            return next;
        });
    };

    const outerItems = [{
        indicator: 1,
        title: 'Подготовка',
        height: '4rem',
        ...outerStatuses[0],
    }, {
        indicator: 2,
        title: 'Установка',
        height: open ? '16rem' : '7rem',
        content: (
            <Accordion stretching="filled">
                <AccordionItem
                    title="Шаги установки"
                    type="arrow"
                    opened={open}
                    onClick={() => setOpen(v => !v)}
                >
                    <Steps
                        orientation="vertical"
                        size="s"
                        hasLine
                        items={innerSteps}
                        onChange={handleInnerChange}
                    />
                </AccordionItem>
            </Accordion>
        ),
        ...outerStatuses[1],
    }, {
        indicator: 3,
        title: 'Проверка',
        ...outerStatuses[2],
    }];

    return (
        <section style={{ width: '400px' }}>
            <Steps orientation="vertical" hasLine items={outerItems} onChange={handleOuterChange} />
        </section>
    );
}
```
