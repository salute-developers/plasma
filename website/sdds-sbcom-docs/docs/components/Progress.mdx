## Примеры

### Вид прогресса

:::caution Устаревшие значения свойства view
Значения `primary`, `success` и `error` устарели. Используйте вместо них:

- `"primary"` → `"accent"`
- `"success"` → `"positive"`
- `"error"` → `"negative"`

Устаревшие значения будут удалены в будущих версиях.
:::

Вид прогресса задается с помощью свойства `view`. Возможные значения свойства `view`:
+ `"default"` – по умолчанию;
+ `"primary"` – основная;
+ `"secondary"` – вторичная;
+ `"accent"` – акцентная;
+ `"accentGradient"` – градиентная;
+ `"info"` – информирующая;
+ `"success"` | `"positive"` – успешное завершение;
+ `"warning"` – предупреждение;
+ `"error"` | `"negative"` – ошибка;

```tsx live
import React from 'react';
import { Progress } from '@salutejs/sdds-sbcom';

export function App() {
    return (
        <div style={{ display: 'flex', flexDirection: 'column', gap: '2rem' }}>
            <Progress value={25} label="Accent" view="accent" hasValue />
            <Progress value={52} label="Positive" view="positive" hasValue />
            <Progress value={40} label="Warning" view="warning" hasValue />
            <Progress value={67} label="Negative" view="negative" hasValue />
        </div>
    );
}
```

### Числовое значение

:::caution Устаревшее свойство displayValue
Свойство `displayValue` устарело. Используйте вместо него `hasValue`.

Устаревшее свойство будет удалено в будущих версиях.
:::

Числовое значение отображается через свойство `hasValue`. Его положение управляется свойством `valuePlacement` — только когда метка не передана:
+ `"top"` – над треком;
+ `"right"` – справа от трека (по умолчанию);
+ `"none"` – значение не отображается.

Когда метка присутствует, положение определяется автоматически из `labelPlacement`.

Когда метка отсутствует и `valuePlacement="top"`, горизонтальное выравнивание значения управляется свойством `valueAlign`:
+ `"start"` – по левому краю (по умолчанию);
+ `"center"` – по центру;
+ `"end"` – по правому краю.

### Метка прогресса

Текстовая метка задаётся через свойство `label`. Её положение относительно трека управляется свойством `labelPlacement`:
+ `"top"` – над треком (по умолчанию). Числовое значение отображается над треком, справа от метки.
+ `"left"` – слева от трека в одну строку. Числовое значение отображается справа от трека.
+ `"none"` – метка не отображается.

При наличии метки положение числового значения определяется автоматически из `labelPlacement` и не зависит от `valuePlacement`.

Дополнительный элемент рядом с текстом метки передаётся через `labelIcon`. Положение текста относительно элемента управляется свойством `labelTextPlacement`:
+ `"right"` – текст справа от иконки (по умолчанию);
+ `"left"` – текст слева от иконки;
+ `"none"` – текст не отображается.

Иконка без текстовой метки — передать `labelIcon` без `label`.

### Высота линии прогресса

Высота линии задаётся через `progressSize` (2, 4, 6, 8 пикселей).

```tsx live
import React from 'react';
import { Progress } from '@salutejs/sdds-sbcom';

export function App() {
    return (
        <div style={{ display: 'flex', flexDirection: 'column', gap: '2rem' }}>
            <Progress value={60} label="2px" progressSize="2" hasValue />
            <Progress value={60} label="4px" progressSize="4" hasValue />
            <Progress value={60} label="6px" progressSize="6" hasValue />
            <Progress value={60} label="8px" progressSize="8" hasValue />
        </div>
    );
}
```

### Подпись

Дополнительная подпись под треком задаётся свойством `caption`.

```tsx live
import React from 'react';
import { Progress } from '@salutejs/sdds-sbcom';

export function App() {
    return (
        <div style={{ display: 'block' }}>
            <Progress value={75} label="Загрузка файла" caption="Осталось 2 минуты" hasValue />
        </div>
    );
}
```
