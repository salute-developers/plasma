## Внешний вид

:::info
Важно! Значения `ItemView` применяются к шагу только когда его статус — `active`.
:::

Внешний вид компонента регулируется 3 свойствами:

#### View

Свойство применяется ко всему компоненту и принимает следующие значения:
- `default`
- `accent`

#### ItemView

Свойство применяется к индикатору активного шага и заголовку, и принимает следующие значения:
- `default`
- `negative`
- `warning`
- `positive`

#### CompletedItemView

Свойство задаётся на каждом шаге отдельно через `items[n].completedItemView` и позволяет переопределить внешний вид индикатора и заголовка для **завершённых** шагов (`status: 'completed'`). Принимает следующие значения:
- `default`
- `accent`
- `negative`
- `warning`
- `positive`

## Тип индикатора

Свойство `indicator` каждого шага определяет, что отображается в индикаторе:

        export function App() {

## Внешнее управление

```tsx live
import React, { useState } from 'react';
import { Steps, Button } from '@salutejs/sdds-finai';

export function App() {
    const items = [{
        indicator: 1,
    }, {
        indicator: 2,
    }, {
        indicator: 3,
    }];

    const [current, setCurrent] = useState<number>(0);
    const onClick = () => setCurrent(() => current + 1);

    return (
        <section>
            <Steps items={items} current={current} />
            <br />
            <Button view="secondary" onClick={onClick}>
                Next
            </Button>
        </section>
    );
}
```

## Использование свойства completedItemView

Свойство `completedItemView` задаётся на каждом шаге индивидуально и позволяет переопределить внешний вид завершённого шага.

```tsx live
import React from 'react';
import { Steps, StepItemProps } from '@salutejs/sdds-finai';

export function App() {
    const items: StepItemProps[] = [{
        indicator: 1,
        title: 'Title',
        content: 'Content',
        status: 'completed',
        completedItemView: 'accent',
    }, {
        indicator: 2,
        title: 'Title',
        content: 'Content',
        status: 'completed',
        completedItemView: 'positive',
    }, {
        indicator: 3,
        title: 'Title',
        content: 'Content',
        status: 'completed',
        completedItemView: 'warning',
    }, {
        indicator: 4,
        title: 'Title',
        content: 'Content',
        status: 'completed',
        completedItemView: 'negative',
    }, {
        indicator: 5,
        title: 'Title',
        content: 'Content',
        status: 'active',
    }];

    return (
        <section>
            <Steps items={items} />
        </section>
    );
}
```
