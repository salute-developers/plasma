## Примеры

### Размер badge
Размер задается с помощью свойства `size`.

Возможные значения свойства: `l`, `m`, `s`, `xs`.

```tsx live
import React from 'react';
import { Badge } from '@salutejs/sdds-finai';

export function App() {
    return (
        <div>
            <Badge text="Бейдж" size="l" />
            <Badge text="Бейдж" size="m" />
            <Badge text="Бейдж" size="s" />
            <Badge text="Бейдж" size="xs" />
        </div>
    );
}
```

### Скругленность badge
Скругленность задается с помощью свойства `pilled`:

```tsx live
import React from 'react';
import { Badge } from '@salutejs/sdds-finai';

export function App() {
    return (
        <div>
            <Badge text="Бейдж" size="l" pilled />
            <Badge text="Бейдж" size="m" pilled />
            <Badge text="Бейдж" size="s" pilled />
            <Badge text="Бейдж" size="xs" pilled />
        </div>
    );
}
```

### Вид badge
Вид `badge` задается с помощью свойства `view`.
Вариант отображения задаётся с помощью свойства `appearance`.

Возможные значения свойства `appearance`: `"default"`, `"clear"`, `"transparent"`.

:::caution Устаревшие свойства
Свойства `transparent` и `clear` устарели. Используйте `appearance="transparent"` и `appearance="clear"` соответственно.
:::

```tsx live
import React from 'react';
import { Badge } from '@salutejs/sdds-finai';

export function App() {
    const Badges = ({ appearance }) => {
        return (
            <div style={{ display: 'flex', gap: '0.5rem' }}>
                <Badge text="Бейдж" size="l" view="default" appearance={appearance} />
                <Badge text="Бейдж" size="l" view="accent" appearance={appearance} />
                <Badge text="Бейдж" size="l" view="positive" appearance={appearance} />
                <Badge text="Бейдж" size="l" view="warning" appearance={appearance} />
                <Badge text="Бейдж" size="l" view="negative" appearance={appearance} />
                <Badge text="Бейдж" size="l" view="dark" appearance={appearance} />
                <Badge text="Бейдж" size="l" view="light" appearance={appearance} />
            </div>
        )
    }

    return (
        <div style={{ display: 'flex', flexDirection: 'column', gap: '2rem' }}>
            <Badges />
            <Badges appearance="transparent" />
            <Badges appearance="clear" />
        </div>
    );
}
```

### Иконка слева / справа
В левой или правой части badge можно отобразить иконку.
Если же нужен Badge с иконкой без текста, нужно использовать `contentLeft`.

```tsx live
import React from 'react';
import { Badge } from '@salutejs/sdds-finai';
import { IconEye } from '@salutejs/plasma-icons';

export function App() {
    return (
        <div>
            <Badge
                text="Бейдж"
                size="l"
                view="default"
                contentLeft={
                    <IconEye color="inherit" size="xs" />
                }
            />
            <Badge
                text="Бейдж"
                size="l"
                view="default"
                contentRight={
                    <IconEye color="inherit" size="xs" />
                }
            />
            <Badge
                size="l"
                view="default"
                contentLeft={
                    <IconEye color="inherit" size="xs" />
                }
            />
        </div>
    );
}
```

### Задание цвета текста и фона
Цвет текста и фона можно задать с помощью свойств
`customColor` и `customBackgroundColor` соответственно.

```tsx live
import React from 'react';
import { Badge } from '@salutejs/sdds-finai';

export function App() {
    return (
        <div>
            <Badge
                text="Бейдж"
                size="l"
                view="default"
                customColor="antiquewhite"
                customBackgroundColor="darkolivegreen"
            />
        </div>
    );
}
```
