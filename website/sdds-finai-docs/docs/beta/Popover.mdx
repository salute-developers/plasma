# Popover
<Badge title="beta" description="Данный компонент входит в группу beta-компонентов" />

:::caution Импорт beta-компонентов
Импорт осуществляется из директории `@salutejs/sdds-finai/beta`<br />
К примеру: <br />
`import * as Beta from '@salutejs/sdds-finai/beta';` <br />
`import { Popover } from '@salutejs/sdds-finai/beta';`
:::

## Описание

Всплывающее окно, привязанное к целевому элементу (<i>target</i>), которое отображает дополнительный контент по клику, наведению или фокусу. Popover идеально подходит для отображения подсказок, контекстных меню и вспомогательных действий.

Popover позволяет отображать всплывающее окно с настраиваемым контентом, позиционированием, триггерами (click, hover, focus), смещением, хвостиком, задержкой открытия/закрытия, ресайзом и стилевыми вариантами.

## Быстрый старт
```tsx live
import * as Beta from '@salutejs/sdds-finai/beta';
import { Button } from '@salutejs/sdds-finai';

export function App() {
    return (
        <Beta.Popover target={<Button text="Target" />}>
            Content
        </Beta.Popover>
    );
}
```

## Взаимодействие с компонентом
Свойство `target` имеет тип `ReactElement`. Это цель для взаимодействия с всплывающим окном.
Содержимое самого окна нужно класть в `children`.

## Стилизация всплывающего окна

Компонент принимает такие свойства, как `style` и `className`, которые применяются к тэгу, оборачивающему выпадающее окно.
Ниже приведен пример кастомизации цвета фона и текста с помощью `styled-components`.

```tsx live
import * as Beta from "@salutejs/sdds-finai/beta";
import { Button } from "@salutejs/sdds-finai";
import {
  inverseTextPrimary,
  inverseSurfaceSolidCardBrightness,
} from "@salutejs/sdds-themes/tokens";
import styled from "styled-components";

export function App() {
  const StyledAsTooltip = styled(Beta.Popover)`
    --plasma-popover-background-color: ${inverseSurfaceSolidCardBrightness};
    color: ${inverseTextPrimary};

    ul, li {
        padding: 0;
        margin: 0;
    }
  `;

  return (
    <StyledAsTooltip target={<Button text="Target" />}>
        <ul>
            <li>Item 1</li>
            <li>Item 2</li>
            <li>Item 3</li>
        </ul>
    </StyledAsTooltip>
  );
}

```
