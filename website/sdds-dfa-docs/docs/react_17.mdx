# Использование библиотеки в React 17 и ниже

## SSRProvider

Для корректной работы приложение нужно обернуть `SSRProvider`;

```jsx
import React from 'react';
import { SSRProvider } from '@salutejs/sdds-dfa';

const App = () => {
    return (
        <SSRProvider>
            ...
        </SSRProvider>
    );
};

export default App;
```
