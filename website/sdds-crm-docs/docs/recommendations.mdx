# Рекомендации

## SSRProvider

Для корректной работы `SSR - server side rendering` приложение нужно обернуть в `SSRProvider`;

```jsx
import React from 'react';
import { SSRProvider } from '@salutejs/sdds-crm';

const App = () => {
    return (
        <SSRProvider>
            ...
        </SSRProvider>
    );
};

export default App;
```
