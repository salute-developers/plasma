## Примеры

### Размер бейджа
Размер задается с помощью свойства `size`. Возможные значения свойтсва: `"l"` или `"s"`.

```tsx live
import React from 'react';
import { Badge } from '@salutejs/plasma-ui';

export function App() {
    return (
        <div>
            <Badge text="Бейдж" size="l" />
            <Badge text="Бейдж" size="s" />
        </div>
    );
}
```

### Вид бейджа
Вид бейджа задается с помощью свойства `view`. Возможные значения свойства `view`:

+ `"primary"` – основной бейдж;
+ `"secondary"` – вторичный бейдж.

```tsx live
import React from 'react';
import { Badge } from '@salutejs/plasma-ui';

export function App() {
    return (
        <div>
            <Badge text="Бейдж" size="l" view="primary" />
            <Badge text="Бейдж" size="l" view="secondary" />
        </div>
    );
}
```

### Иконка слева
В левой части бейджа можно отобразить иконку.
Если не указывать текст, бейдж будет содержать только заданную иконку:

```tsx live
import React from 'react';
import { Badge } from '@salutejs/plasma-ui';
import { IconDownload } from '@salutejs/plasma-icons';

export function App() {
    return (
        <div>
            <Badge text="Бейдж" size="l" view="primary" contentLeft={<IconDownload color="inherit" size="xs" />} />
            <Badge size="l" view="primary" contentLeft={<IconDownload color="inherit" size="xs" />} />
        </div>
    );
}
```

### Градиентный текст
Текст бейджа можно окрасить в градиентный цвет с помощью класса `sdds-core-text-node`:

```tsx live
import React from 'react';
import { Badge } from '@salutejs/plasma-ui';
import styled from 'styled-components';

export function App() {
    const GradientBadge = styled(Badge)`
        & .sdds-core-text-node {
            background: linear-gradient(90deg, #f7971e, #ffd200);
            background-clip: text;
            -webkit-background-clip: text;
            -webkit-text-fill-color: transparent;
    }
    `;
    return (
        <div style={{ display: 'flex', gap: '1rem', flexWrap: 'wrap' }}>
            <GradientBadge
                text="Бейдж"
            />
        </div>
    );
}
```
