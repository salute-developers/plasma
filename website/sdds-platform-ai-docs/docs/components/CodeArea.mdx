# CodeArea

Компонент `CodeArea` используется для отображения фрагментов кода с подсветкой синтаксиса.

<Description name="CodeArea" />

## Быстрый старт

Код передается в `children`, а язык подсветки задается свойством `language`.

```tsx live
import React from 'react';
import { CodeArea } from '@salutejs/sdds-platform-ai';

export function App() {
    return (
        <CodeArea language="json" showLineNumbers>
            {`{
  "input": "Первый искусственный спутник Земли",
  "model": "GigaCheckClassification"
}`}
        </CodeArea>
    );
}
```

## Номера строк

Свойство `showLineNumbers` включает колонку с номерами строк. Ширина колонки рассчитывается по номеру последней строки,
поэтому однозначные и многозначные номера не меняют ширину строки кода.

```tsx live
import React from 'react';
import { CodeArea } from '@salutejs/sdds-platform-ai';

export function App() {
    return (
        <CodeArea language="javascript" showLineNumbers>
            {`const model = 'GigaCheckClassification';
const input = 'Первый искусственный спутник Земли';
const enabled = true;`}
        </CodeArea>
    );
}
```

## Перенос длинных строк

Длинные строки переносятся автоматически и остаются внутри области кода.

```tsx live
import React from 'react';
import { CodeArea } from '@salutejs/sdds-platform-ai';

export function App() {
    return (
        <div style={{ width: '22rem' }}>
            <CodeArea language="javascript" showLineNumbers>
                {`const message = 'This is a very long line that wraps inside CodeArea without changing the line number column width.';`}
            </CodeArea>
        </div>
    );
}
```

## Appearance

Свойство `appearance` переключает вариант отображения компонента. По умолчанию используется `default`.
Для варианта с внешней оберткой используйте `appearance="hasWrapper"`.

```tsx live
import React from 'react';
import { CodeArea } from '@salutejs/sdds-platform-ai';

export function App() {
    return (
        <CodeArea appearance="hasWrapper" language="json" showLineNumbers>
            {`{
  "status": "ok",
  "source": "CodeArea"
}`}
        </CodeArea>
    );
}
```

## Слоты

Через `contentOuter`, `contentHeader` и `contentAction` можно передать дополнительный контент вокруг области кода.
Свойство `hasDivider` добавляет разделитель снизу header-слота.

```tsx live
import React from 'react';
import { CodeArea } from '@salutejs/sdds-platform-ai';
import { IconCopyOutline } from '@salutejs/plasma-icons';
import styled from 'styled-components';

export function App() {
    const slotStyle = {
        display: 'flex',
        alignItems: 'center',
        justifyContent: 'center',
        height: '3rem',
        background:
            'linear-gradient(to right, #9747FF 0.063rem, transparent 0.063rem) 0 0, linear-gradient(to right, #9747FF 0.063rem, transparent 0.063rem) 0 100%, linear-gradient(to left, #9747FF 0.063rem, transparent 0.063rem) 100% 0, linear-gradient(to left, #9747FF 0.063rem, transparent 0.063rem) 100% 100%, linear-gradient(to bottom, #9747FF 0.063rem, transparent 0.063rem) 0 0, linear-gradient(to bottom, #9747FF 0.063rem, transparent 0.063rem) 100% 0, linear-gradient(to top, #9747FF 0.063rem, transparent 0.063rem) 0 100%, linear-gradient(to top, #9747FF 0.063rem, transparent 0.063rem) 100% 100%',
        backgroundRepeat: 'no-repeat',
        backgroundSize: '1rem 1rem',
    };

    return (
        <div style={{ width: '640px' }}>
            <CodeArea
                appearance="hasWrapper"
                contentAction={<IconCopyOutline color="inherit" size="s" />}
                contentHeader={<div style={slotStyle}>Header</div>}
                contentOuter={<div style={slotStyle}>Content</div>}
                hasDivider
                language="json"
                showLineNumbers
            >
                {`{
  "status": "ok",
  "source": "CodeArea"
}`}
            </CodeArea>
        </div>
    );
}
```
