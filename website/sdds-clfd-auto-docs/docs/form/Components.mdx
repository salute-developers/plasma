# Поддержка компонентов 

### TextField, TextArea, Radiobox, Checkbox, Autocomplite и Switch

#### React Hook Form

Изначальные значения задаются в defaultValues. Возвращаемый и изначальный тип как у нативных элементов.

#### Нативная форма 

Изначальные значения задаются в defaultValue через параметр в компоненте.


### Single Slider

#### React Hook Form

Изначальные значения задаются в defaultValue через параметр в компоненте. Возвращаемый и изначальный тип `number`.

#### Нативная форма 

Изначальные значения задаются в defaultValue через параметр в компоненте. Возвращаемый и изначальный тип `number`.


### Double Slider

#### React Hook Form

Изначальные значения задаются в defaultValue (`number[]`) через параметр в компоненте. Возвращаемый тип `string` в формате `[MIN] - [MAX]`.

#### Нативная форма 

Изначальные значения задаются в defaultValue (`number[]`) через параметр в компоненте. Возвращается два ключа с одинаковым именем. Первое значение `min`, второе `max`. 
```

[
    ['textfield': 'test'],
    ...
    ['sliderdouble': 1],
    ['sliderdouble': 3],
    ...
    ['textfield2': 'test'],
]

```


### DatePicker

#### Нативная форма 

Изначальные значения задаются в defaultValue через параметр в компоненте. Возвращаемый и изначальный тип `string`.


### DatePicker Range

#### Нативная форма 

Изначальные значения задаются в defaultValue (`string[]`) через параметр в компоненте. Первое значение `min`, второе `max`. 

```

[
    ['textfield': 'test'],
    ...
    ['datepickerRange': '01.01.2024'],
    ['datepickerRange': '11.01.2024'],
    ...
    ['textfield2': 'test'],
]

```
