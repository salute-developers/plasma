# Pagination
<Description name="Pagination" />

## Быстрый старт

Свойство `slots` задаёт максимальное количество кнопок страниц, которые отображаются одновременно.
Свойство `count` задаёт общее количество элементов (или страниц, если `hasPerPage` не указан):

```tsx live
import React from 'react';
import { Pagination } from '@salutejs/sdds-cs';

export function App() {
    return (
        <div>
            <Pagination slots={9} count={2000} hasPerPage perPage={20} />
        </div>
    );
}
```

## Тип

С помощью свойства `type` можно выбрать тип отображения компонента:

- `"default"` — обычный, с отображением номеров страниц;
- `"compact"` — компактный, показывает только текущую страницу.

```tsx live
import React from 'react';
import { Pagination } from '@salutejs/sdds-cs';

export function App() {
    return (
        <div style={{ display: 'flex', gap: '1rem', flexDirection: 'column' }}>
            <Pagination size="s" type="default" slots={9} count={200} hasPerPage />
            <Pagination size="s" type="compact" slots={9} count={200} hasPerPage />
        </div>
    );
}
```

## Форма кнопок

С помощью свойства `pilled` можно переключить кнопки в круглую форму:

```tsx live
import React from 'react';
import { Pagination } from '@salutejs/sdds-cs';

export function App() {
    return (
        <div style={{ display: 'flex', gap: '1rem', flexDirection: 'column' }}>
            <Pagination size="s" pilled={false} slots={9} count={200} />
            <Pagination size="s" pilled={true} slots={9} count={200} />
        </div>
    );
}
```

## Количество элементов на странице

С помощью свойства `hasPerPage` можно включить выпадающий список выбора количества элементов на странице. Начальное значение задаётся свойством `perPage`:

```tsx live
import React from 'react';
import { Pagination } from '@salutejs/sdds-cs';

export function App() {
    return (
        <div style={{ display: 'flex', gap: '1rem', flexDirection: 'column' }}>
            <Pagination size="s" slots={9} count={200} />
            <Pagination size="s" slots={9} count={200} hasPerPage perPage={20} />
        </div>
    );
}
```

## Быстрый переход на страницу

С помощью свойства `hasQuickJump` можно включить поле быстрого перехода на произвольную страницу:

```tsx live
import React from 'react';
import { Pagination } from '@salutejs/sdds-cs';

export function App() {
    return (
        <div style={{ display: 'flex', gap: '1rem', flexDirection: 'column' }}>
            <Pagination size="s" slots={9} count={200} />
            <Pagination size="s" slots={9} count={200} hasQuickJump />
        </div>
    );
}
```

## Вспомогательный текст

С помощью свойства `helperText` можно добавить произвольный текст рядом с пагинацией.
Свойство `singleLine` ограничивает текст одной строкой с многоточием:

```tsx live
import React from 'react';
import { Pagination } from '@salutejs/sdds-cs';

export function App() {
    return (
        <div style={{ display: 'flex', gap: '1rem', flexDirection: 'column' }}>
            <Pagination size="s" slots={9} count={200} helperText="Страница 1 из 10" />
            <Pagination
                size="s"
                slots={9}
                count={200}
                helperText="Очень длинный вспомогательный текст, который может не поместиться в одну строку"
                singleLine
            />
        </div>
    );
}
```

## Навигационные кнопки

С помощью свойств `leftContent` и `rightContent` можно разместить произвольный контент по бокам от блока страниц.
Например, кнопки навигации к первой, предыдущей, следующей и последней страницам:

```tsx live
import React, { useState } from 'react';
import { Pagination, IconButton } from '@salutejs/sdds-cs';
import {
    IconDoubleDisclosureLeft,
    IconDisclosureLeftOutline,
    IconDisclosureRightOutline,
    IconDoubleDisclosureRight,
} from '@salutejs/plasma-icons';

export function App() {
    const count = 2000;
    const perPage = 20;
    const [page, setPage] = useState(1);

    const lastPage = Math.ceil(count / perPage);

    const leftContent = (
        <>
            <IconButton view="clear" size="s" onClick={() => setPage(1)}>
                <IconDoubleDisclosureLeft color="inherit" size="s" />
            </IconButton>
            <IconButton view="clear" size="s" onClick={() => setPage((p) => Math.max(1, p - 1))}>
                <IconDisclosureLeftOutline color="inherit" size="s" />
            </IconButton>
        </>
    );

    const rightContent = (
        <>
            <IconButton view="clear" size="s" onClick={() => setPage((p) => Math.min(lastPage, p + 1))}>
                <IconDisclosureRightOutline color="inherit" size="s" />
            </IconButton>
            <IconButton view="clear" size="s" onClick={() => setPage(lastPage)}>
                <IconDoubleDisclosureRight color="inherit" size="s" />
            </IconButton>
        </>
    );

    return (
        <div>
            <Pagination
                size="s"
                slots={9}
                count={count}
                perPage={perPage}
                value={page}
                leftContent={leftContent}
                rightContent={rightContent}
                onChange={(newPage) => newPage !== undefined && setPage(newPage)}
            />
        </div>
    );
}
```

## Контролируемый режим

Компонент поддерживает контролируемый режим через свойства `value` и `perPage`.
Колбэк `onChange` вызывается с новым номером страницы и текущим количеством элементов на странице:

```tsx live
import React, { useState } from 'react';
import { Pagination } from '@salutejs/sdds-cs';

export function App() {
    const [page, setPage] = useState(1);
    const [perPage, setPerPage] = useState(20);

    return (
        <div style={{ display: 'flex', gap: '1rem', flexDirection: 'column' }}>
            <Pagination
                size="s"
                slots={9}
                count={2000}
                value={page}
                perPage={perPage}
                hasPerPage
                hasQuickJump
                onChange={(newPage, newPerPage) => {
                    if (newPage !== undefined) setPage(newPage);
                    if (newPerPage !== undefined) setPerPage(newPerPage);
                }}
            />
            <div>Страница: {page}, элементов на странице: {perPage}</div>
        </div>
    );
}
```
