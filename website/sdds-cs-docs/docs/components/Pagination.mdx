# Pagination

Pagination может отображаться в разных размерах и может содержать select с выбором количества элементов на странице и быстрый выбор страницы

## Pagination

## Использование
Компонент `Pagination` должен содержать параметр slots. Он обозначает количество кнопок в Pagination

Также компонент имеет параметр hasPerPage, логика работы такова:
Если указан только параметр count, то он равен кол-ву страниц
Если указан параметр hasPerPage или perPage, то count делится на perPage

```tsx live
import React from 'react';
import { Pagination } from '@salutejs/sdds-cs';

export function App() {
    return (
        <div style={{display: "flex", "flex-direction": "column"}}>
            <div>
                <Pagination slots={7} count={200} />
            </div>
            <div>
                <Pagination slots={9} count={200} hasPerPage={true} perPage={20} />
            </div>
            <div>
                <Pagination slots={9} count={200} hasPerPage={true} />
            </div>
            <div>
                <Pagination slots={9} count={200} hasPerPage={false} />
            </div>
        </div>
    );
}
```

## Примеры

### Размер Pagination
Размер Pagination задается с помощью свойства `size`. Возможные значения свойства: `"l"`, `"m"`, `"s"` или `"xs"`:

```tsx live
import React from 'react';
import { Pagination } from '@salutejs/sdds-cs';

export function App() {
    return (
        <div style={{display: "flex", "flex-direction": "column"}}>
            <Pagination size="s" slots={9} count={200} hasPerPage={true} />
        </div>
    );
}
```

### Типы Pagination
Тип Pagination задается с помощью свойства `type`. Возможные значения свойства `type`:
+ `"default"` – обычная;
+ `"compact"` – компактная;

```tsx live
import React from 'react';
import { Pagination } from '@salutejs/sdds-cs';

export function App() {
    return (
        <div style={{display: "flex", "flex-direction": "column"}}>
            <Pagination size="s" type="default" slots={9} count={200} hasPerPage={true} />
            <Pagination size="s" type="compact" slots={9} count={200} hasPerPage={true} />
        </div>
    );
}
```
