# Steps

## Тип индикатора

Свойство `indicator` каждого шага определяет, что отображается в индикаторе:

<Tabs>
    <TabItem value="numbered" label="Numbered" default>
        ```tsx live
        import React from 'react';
        import { Steps } from '@salutejs/plasma-homeds';

        export function App() {
            const initialItems = [{
                indicator: 1,
                title: 'Title',
                content: 'Content',
            }, {
                indicator: 2,
                title: 'Title',
                content: 'Content',
            }, {
                indicator: 3,
                title: 'Title',
                content: 'Content',
            }];

            const [items, setItems] = useState(initialItems);

            const onChange = (item: StepItemProps, index: number, prevIndex: number) => {
                if (prevIndex !== undefined) {
                    items[prevIndex].status = 'completed';
                }

                items[index].status = 'active';

                setItems([...items]);
            };

            return (
                <section>
                    <Steps items={items} onChange={onChange} />
                </section>
            );
        }
        ```

        Свойство `indicator` также принимает render-функцию, которая получает `{ status, size }` и возвращает иконку.

        ```tsx live
        import React from 'react';
        import { Steps } from '@salutejs/plasma-homeds';
        import { IconPlasma } from '@salutejs/plasma-icons';

        export function App() {
            const renderIndicator = ({ status, size }) => {
                const iconSize = size === 'xs' || size === 's' ? 'xs' : 'xs';
                return <IconPlasma color="inherit" size={iconSize} />;

                content: 'Content',
        ```

## Вариант использования с управлением текущим элементом из вне

```tsx live
import React, { useState } from 'react';
import { Steps, Button } from '@salutejs/sdds-cs';

export function App() {
    const items = [{
        indicator: 1,
    }, {
        indicator: 2,
    }, {
        indicator: 3,
    }];

    const [current, setCurrent] = useState<number>(0);
    const onClick = () => setCurrent(() => current + 1);

    return (
        <section>
            <Steps items={items} current={current} />
            <br />
            <Button view="secondary" onClick={onClick}>
                Next
            </Button>
        </section>
    );
}
```
