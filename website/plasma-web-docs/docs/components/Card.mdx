## Примеры

### Простая карточка
Представляет собой ответ в виде карточки, которая может содержать ячейку с изображением.
Включает в себя класс карточек нескольких видов.

```tsx live
import React from 'react';
import { 
    Card, 
    CardBody, 
    CardMedia, 
    CardBadge, 
    CardContent,
    Body1,
    Headline4
} from '@salutejs/plasma-web';

export function App() {
    const src = 'https://plasma.sberdevices.ru/ui/images/320_320_1.jpg';

    return (
        <div>
            <Card style={{ width: '22.5rem' }} tabIndex={0} outlined scaleOnFocus>
                <CardBody>
                    <CardMedia src={src} placeholder={src} ratio="1/1" />
                    <CardBadge style={{ left: '1rem', top: '1rem' }} text="Test in badge" />
                    <CardContent>
                        <Body1>Label</Body1>
                        <Headline4>Tittle</Headline4>
                        <Body1>description</Body1>
                    </CardContent>
                </CardBody>
            </Card>
        </div>
    );
}
```
