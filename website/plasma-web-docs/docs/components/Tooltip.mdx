## Базовое применение

`Tooltip` оборачивает собой любой компонент и открывается в заданную сторону.

Предполагается оборачивать иконки из пакета `@salutejs/plasma-icons`

```tsx live
<Tooltip text="Высокое качество воспроизведения" visible>
    <IconApps />
</Tooltip>
```
