## Примеры

### Ориентация разделителя
Ориентация разделителя задается с помощью свойства orientation. Возможные значения свойства orientation:
+ "horizontal" – горизонтальная (по умолчанию);
+ "vertical" – вертикальная;

```tsx live

import React from 'react';
import { BodyS, Divider } from '@salutejs/plasma-web';

export function App() {
    const Demo = ({
        orientation = 'horizontal',
        orientationScale = '100%',
        beforeText = 'Before',
        afterText = 'After',
    }) => {
        const wrapperStyle = {
            display: "inline-flex",
            gap: "0.5rem",
            alignItems: "center",
            justifyContent: "center",
            flexDirection: orientation === "horizontal" ? "column" : "row"
        }

        return (
            <div style={wrapperStyle}>
                <BodyS>{beforeText}</BodyS>
                <Divider orientation={orientation} orientationScale={orientationScale} />
                <BodyS>{afterText}</BodyS>
            </div>
        );
    };

    return (
        <div style={{ display: "flex", gap: "2rem" }}>
            <Demo />
            <Demo orientation="vertical" />
        </div>
    );
}
```
