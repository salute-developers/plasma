## Примеры
### Подчёркивание

Prop `underline` управляет режимом подчёркивания ссылки:

- `none` — без подчёркивания (по умолчанию)
- `hover` — подчёркивание при наведении
- `always` — всегда подчёркнута

```tsx live
import React from 'react';
import { P1, Link } from '@salutejs/plasma-web';

export function App() {
    return (
        <div style={{ display: 'flex', flexDirection: 'column', gap: '8px' }}>
            <P1><Link href="/" underline="none">Без подчёркивания</Link></P1>
            <P1><Link href="/" underline="hover">Подчёркивание при наведении</Link></P1>
            <P1><Link href="/" underline="always">Всегда подчёркнута</Link></P1>
        </div>
    );
}
```
