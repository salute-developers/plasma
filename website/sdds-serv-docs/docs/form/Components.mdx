# Поддержка компонентов 

### TextField и TextArea

#### Hook UseForm 

Изначальные значения задаются в defaultValues. Возвращаемый и изначальный тип `string`.

#### React Hook Form

Изначальные значения задаются в defaultValues. Возвращаемый и изначальный тип `string`.

#### Нативная форма 

Изначальные значения задаются в defaultValue через параметр в компоненте. Возвращаемый и изначальный тип `string`.


### Checkbox и Switch

#### Hook UseForm 

Изначальные значения задаются в defaultValues. Возвращаемый и изначальный тип `boolean`.

#### React Hook Form

Изначальные значения задаются в defaultValues. Возвращаемый и изначальный тип `boolean`.

#### Нативная форма 

Изначальные значения задаются в defaultValue / value через параметр в компоненте. Возвращаемый и изначальный тип `boolean`.

#### Нативная форма

### Single Slider

#### Hook UseForm 

Изначальные значения задаются в defaultValues. Возвращаемый и изначальный тип `number`.

#### React Hook Form

Изначальные значения задаются в defaultValue через параметр в компоненте. Возвращаемый и изначальный тип `number`.

#### Нативная форма 

Изначальные значения задаются в defaultValue через параметр в компоненте. Возвращаемый и изначальный тип `number`.


### Double Slider

#### Hook UseForm 

Изначальные значения задаются в defaultValues. Возвращаемый и изначальный тип `number[]`.

#### React Hook Form

Изначальные значения задаются в defaultValue (`number[]`) через параметр в компоненте. Возвращаемый тип `string` в формате `[MIN] - [MAX]`.

#### Нативная форма 

Изначальные значения задаются в defaultValue (`number[]`) через параметр в компоненте. Возвращается два ключа с одинаковым именем. Первое значение `min`, второе `max`. 
```

[
    textfield: 'test',
    ...
    sliderdouble: 1,
    sliderdouble: 3,
    ...
    textfield2: 'test',
]

```

### DatePicker Range

#### Hook UseForm 

Изначальные значения задаются в defaultValues. Возвращаемый и изначальный тип `string[]`.

#### React Hook Form

Изначальные значения задаются в defaultValue (`string[]`) через параметр в компоненте. Возвращаемый тип `string` в формате `[MIN] - [MAX]`.

#### Нативная форма 

Изначальные значения задаются в defaultValue (`string[]`) через параметр в компоненте. Первое значение `min`, второе `max`. 

```

[
    textfield: 'test',
    ...
    datepickerRange: '01.01.2024',
    datepickerRange: '11.01.2024',,
    ...
    textfield2: 'test',
]

```
