## Использование
Компонент `Rating` представляет собой неинтерактивный компонент для отображения оценки.

```tsx live
import React from 'react';
import { Rating } from '@salutejs/sdds-serv';

export function App() {
    return (
        <div>
            <Rating
                view="default"
                size="l"
                value={3.8}
                helperText="Helper text"
            />
        </div>
    );
}
```

### Вид Rating
Вид Rating задаётся с помощью свойства `view`:

```tsx live
import React from 'react';
import { Rating } from '@salutejs/sdds-serv';

export function App() {
    return (
        <div style={{ display: "flex", flexDirection: "column", gap: "2rem" }}>
            <Rating
                view="default"
                size="l"
                value={3.8}
                helperText="Helper text"
            />
            <Rating
                view="accent"
                size="l"
                value={3.8}
                helperText="Helper text"
            />
        </div>
    );
}
```

### Количество иконок в Rating

```tsx live

```

### Положение значения в Rating
Положение значения в Rating задается с помощью свойства `valuePlacement`. 
Отображение значения можно выключить с помощью свойства `hasValue`:

```tsx live
import React from 'react';
import { Rating } from '@salutejs/sdds-serv';

export function App() {
    return (
        <div style={{ display: "flex", flexDirection: "column", gap: "2rem" }}>
            <Rating
                view="default"
                size="l"
                value={3.8}
                valuePlacement="before"
                helperText="Helper text"
            />
            <Rating
                view="accent"
                size="l"
                value={3.8}
                valuePlacement="after"
                helperText="Helper text"
            />
            <Rating
                view="accent"
                size="l"
                value={3.8}
                hasValue={false}
                helperText="Helper text"
            />
        </div>
    );
}
```

### Пользовтельские иконки в Rating
Пользовтельские иконки в Rating задаются с помощью свойств:
+ `iconSlot` – закрашенная иконка;
+ `iconSlotOutline` – outline иконка;
+ `iconSlotHalf` - иконка для отображения дробного значения оценки.

Для корректного применения размеров к иконкам, необходимо передавать класс `ratingClasses.customIconSizing` ("custom-icon-sizing")


```tsx live
import React from 'react';
import { IconKeyFill, IconKeyOutline, IconLockFill } from '@salutejs/plasma-icons';
import { ratingClasses, Rating } from '@salutejs/sdds-serv';

export function App() {
    return (
        <div style={{ display: "flex", flexDirection: "column", gap: "2rem" }}>
            <Rating
                view="default"
                size="l"
                value={3.8}
                helperText="Helper text"
                iconSlot={<IconKeyFill size="s" className={ratingClasses.customIconSizing} />}
                iconSlotOutline={<IconLockFill size="s" className={ratingClasses.customIconSizing} />}
                iconSlotHalf={<IconKeyOutline size="s" className={ratingClasses.customIconSizing} />}
            />
        </div>
    );
}
```
