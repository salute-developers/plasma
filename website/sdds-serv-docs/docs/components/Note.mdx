# Note
Компонент для отображения текста в виде информационных, предупреждающих, обычных подсказок.

<Description name="Note" />
<PropsTable name="Note" />

## Примеры

<Tabs>

                        size="s"
                        contentBefore={<IconSaluteOutline size="xs" color="inherit" />}

    </TabItem>

    <TabItem value="size" label="Size">
        Размер компонента задается с помощью свойства `size`.\
        Также можно указать собственную ширину и высоту с помощью `width` и `height` соответственно.\
        Или задать `stretch`, чтобы `Note` занял всю доступную ширину и высоту.

        ```tsx live
        import React from 'react';
        import { Note } from '@salutejs/sdds-serv';

        export function App() {
            return (
                <div style={{ display: 'flex', flexDirection: 'column', gap: '1rem' }}>
                    <div style={{ display: 'flex', gap: '1rem', flexWrap: 'wrap' }}>
                        <Note title="Title" text="Text" view="default" size="l" />
                        <Note title="Title" text="Text" view="default" size="m" />
                        <Note title="Title" text="Text" view="default" size="s" />
                        <Note title="Title" text="Text" view="default" size="xs" />
                    </div>
                    <Note
                        title="Title"
                        text="Text"
                        view="default"
                        size="l"
                        width="400"
                        height="116"
                    />
                    <div style={{ width: '300px', height: '300px' }}>
                        <Note title="Title" text="Text" view="default" size="l" stretch />
                    </div>
                </div>
            );
        }
        ```
    </TabItem>

    <TabItem value="orientation" label="Orientation">
        Ориентация компонента задается с помощью свойства `orientation`.\
        При значении `horizontal` контент выстраивается в строку: иконка, заголовок и текст слева, `actionContent` и кнопка закрытия справа.

        ```tsx live
        import React from 'react';
        import { Note, LinkButton } from '@salutejs/sdds-serv';
        import { IconSaluteOutline } from '@salutejs/plasma-icons';

        export function App() {
            return (
                <div style={{ display: 'flex', flexDirection: 'column', gap: '1rem' }}>
                    <Note
                        title="Title"
                        text="Text"
                        view="default"
                        size="l"
                        orientation="vertical"
                        contentBefore={<IconSaluteOutline size="s" color="inherit" />}
                    />
                    <Note
                        title="Title"
                        text="Text"
                        view="default"
                        size="l"
                        orientation="horizontal"
                        contentBefore={<IconSaluteOutline size="s" color="inherit" />}
                        actionContent={<LinkButton text="Label" />}
                        hasClose
                    />
                </div>
            );
        }
        ```

### Контент `contentBefore`

Размерность и положение `contentBefore` задается с помощью свойства `contentBeforeSizing`.

```tsx live
import React from 'react';
import { Note } from '@salutejs/sdds-serv';
import { IconSaluteOutline } from '@salutejs/plasma-icons';

export function App() {
    return (
        <div style={{ display: 'flex', flexDirection: 'column', gap: '1rem' }}>
            <Note
                title="Title"
                text="Text"
                view="default"
                size="s"
                contentBefore={<IconSaluteOutline size="xs" color="inherit" />}
                contentBeforeSizing="fixed"
            />
            <Note
                title="Title"
                text="Text"
                view="positive"
                size="s"
                contentBefore={<IconSaluteOutline size="m" color="inherit" />}
                contentBeforeSizing="scalable"
            />
        </div>
    );
}
```

### Дополнительный контент `actionContent`

Свойство `actionContent` позволяет разместить произвольный контент (например, кнопку) после основного текста.

```tsx live
import React from 'react';
import { Note, LinkButton } from '@salutejs/sdds-serv';
import { IconSaluteOutline } from '@salutejs/plasma-icons';

export function App() {
    return (
        <div style={{ display: 'flex', flexDirection: 'column', gap: '1rem' }}>
            <Note
                title="Title"
                text="Text"
                view="default"
                size="l"
                contentBefore={<IconSaluteOutline size="s" color="inherit" />}
                actionContent={<LinkButton text="Label" />}
            />
            <Note
                title="Title"
                text="Text"
                view="positive"
                size="l"
                orientation="horizontal"
                contentBefore={<IconSaluteOutline size="s" color="inherit" />}
                actionContent={<LinkButton text="Label" />}
            />
        </div>
    );
}
```

### Кнопка закрытия `hasClose`

Свойство `hasClose` добавляет кнопку закрытия. При нажатии срабатывает функция `onCloseButtonClick`.

```tsx live
import React from 'react';
import { Note, Button } from '@salutejs/sdds-serv';
import { IconSaluteOutline } from '@salutejs/plasma-icons';

export function App() {
  const [state, setState] = useState(true);

  return (
    <div style={{ height: '8rem', display: 'block' }}>
      <Button
        size="s"
        style={{ marginBottom: '0.5rem' }}
        onClick={() => setState(!state)}
      >
        Открыть
      </Button>
      {state && (
        <Note
          title="Ваш заголовок"
          text="Пример с возможностью закрыть по control"
          view="default"
          size="s"
          contentBefore={<IconSaluteOutline size="xs" color="inherit" />}
          hasClose
          onCloseButtonClick={() => setState(false)}
        />
      )}
    </div>
  );
}
```
